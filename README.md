# Shreyan Dev - Portfolio

A modern, responsive developer portfolio built with React and Tailwind CSS. Features a clean, minimal design with smooth animations and a single-scroll layout optimized for showcasing professional work.

## Tech Stack

- **React 18** - Modern UI framework
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **shadcn/ui** - Component library
- **Vite** - Build tool and dev server
- **Framer Motion** - Smooth animations

## Features

- **Responsive Design** - Mobile-first approach with breakpoint optimization
- **Accessibility** - WCAG compliant with semantic HTML
- **SEO Optimized** - Meta tags, structured data, and performance
- **Modern UI** - Clean, professional aesthetic with smooth transitions
- **Component Architecture** - Modular, reusable components
- **Performance** - Optimized bundle size and loading times

## Project Structure

```
src/
├── components/          # Reusable UI components
│   ├── ui/             # shadcn/ui components
│   └── ...             # Custom portfolio components
├── data/               # Static content and configuration
├── hooks/              # Custom React hooks
├── lib/                # Utility functions
└── pages/              # Page components
```

## Development

Built with modern web technologies for optimal performance and developer experience. The codebase emphasizes maintainability, scalability, and clean architecture patterns.
