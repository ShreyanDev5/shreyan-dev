# Shreyan Sardar | Portfolio Website

A personal developer portfolio website showcasing backend engineering skills and interactive full-stack projects.

## 🚀 Tech Stack

- **Framework & Runtime**: React 18, TypeScript, Vite
- **Styling & Animation**: Tailwind CSS, Framer Motion
- **Icons & Routing**: Lucide React, React Icons, React Router DOM

## ✨ Core Features

- **Terminal Mockup**: Interactive CLI simulator to query biography, projects, and contact info.
- **Project Showcase**: Filterable project cards with links, details, and demo login credentials.
- **Journey Timeline**: Chronological display of career milestones, skills, and academic history.
- **Contact Info**: Responsive layout containing social links and an email copy-to-clipboard shortcut.
